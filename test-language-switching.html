üêõ Troubleshooting

If language doesn't change:

  1. Clear browser cache and localStorage:
    localStorage.clear();
    location.reload();
  2. Check for console errors
  3. Verify translation files exist in src/i18n/locales/
  4. Ensure all components use useTranslation() hook

If page doesn't reload:

  1. Check InlineLanguageSelector.jsx has window.location.reload()
  2. Check browser console for JavaScript errors
  3. Try manual reload after language change
