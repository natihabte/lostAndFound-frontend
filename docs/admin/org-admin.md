# Organization Admin Guide

Complete guide for Organization Administrators.

## Overview

Organization Admins manage their specific organization with control over:
- Organization settings and branding
- User management within organization
- Item oversight and approval
- Claim processing
- Organization-specific reports

## Access

**Login URL**: `/admin/org-login`

Credentials provided by Super Admin upon organization approval.

## Dashboard

Organization Admin dashboard shows:
- Organization statistics
- Recent items and claims
- Pending approvals
- User activity
- Quick actions

## Organization Management

### Profile Settings

Update organization information:
- Name and description
- Contact information
- Address and locations
- Operating hours
- Website and social media

### Branding

Customize organization appearance:
- Upload logo
- Set color scheme
- Custom CSS (if enabled)
- Email template customization

### Locations

Manage organization locations:
- Add new locations
- Edit location details
- Assign hall admins
- Set location-specific settings

## User Management

### Manage Users

- View all organization users
- Search and filter users
- Verify user accounts
- Suspend or remove users
- Export user lists

### Hall Admin Management

- Create hall admin accounts
- Assign locations to hall admins
- Set permissions
- Monitor hall admin activity

### User Roles

Assign roles within organization:
- Regular users
- Hall admins
- Organization admins (with Super Admin approval)

## Item Management

### Review Items

- View all items in organization
- Approve or reject items
- Edit item details
- Mark items as returned
- Delete inappropriate items

### Item Categories

- Customize categories for organization
- Set category-specific rules
- Monitor category usage

### Approval Workflow

Configure item approval process:
- Automatic approval
- Manual approval required
- Trusted user auto-approval

## Claim Management

### Process Claims

- Review pending claims
- Verify proof of ownership
- Approve or reject claims
- Coordinate item returns
- Close completed claims

### Claim Verification

- Set verification requirements
- Request additional documentation
- Contact claimants and finders
- Resolve disputes

## Reports and Analytics

### Organization Statistics

- User engagement
- Item statistics
- Claim success rates
- Popular categories
- Location performance

### Custom Reports

- Generate custom reports
- Export data
- Schedule reports
- Share with stakeholders

## Settings

### Notification Settings

- Configure email notifications
- Set notification frequency
- Customize notification content

### Privacy Settings

- Data retention policies
- User data visibility
- Search settings
- Public access controls

### Integration Settings

- API access
- SSO configuration
- External system integration

## Best Practices

1. **Regular Reviews**: Check pending items and claims daily
2. **User Support**: Respond to user inquiries promptly
3. **Data Quality**: Maintain accurate organization information
4. **Training**: Provide training for hall admins
5. **Communication**: Keep users informed of policies

Organization Admins ensure smooth operation of their organization's lost and found system.
