# User Management

Comprehensive guide for managing users across the platform.

## Overview

User management capabilities vary by admin level:
- **Super Admin**: All users across all organizations
- **Organization Admin**: Users within their organization
- **Hall Admin**: Limited user support functions

## User Roles

### Role Hierarchy

1. **Super Admin** - Platform-wide control
2. **Organization Admin** - Organization-level control
3. **Hall Admin** - Location-level control
4. **User** - Standard platform access

### Permissions by Role

**Super Admin:**
- Manage all users
- Assign any role
- Access all organizations
- System configuration

**Organization Admin:**
- Manage organization users
- Assign hall admin roles
- Organization settings
- Organization reports

**Hall Admin:**
- View location users
- Assist with user issues
- Limited user support

**User:**
- Report items
- Submit claims
- Manage profile
- Search items
