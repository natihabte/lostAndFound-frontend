# Item Management Guide

Learn how to effectively manage lost and found items on the platform.

## Overview

The item management system allows users to report lost items, report found items, and manage their submissions throughout the lifecycle of each item.

## Reporting Lost Items

### Step-by-Step Process

1. **Navigate to Report Lost Item**
   - Click "Report Lost Item" from the main menu
   - Or use the quick action button on the dashboard

2. **Fill in Item Details**
   - **Item Name**: Clear, descriptive name
   - **Category**: Select appropriate category
   - **Description**: Detailed description including:
     - Color, size, brand
     - Distinguishing features
     - Serial numbers (if applicable)
   - **Location**: Where the item was lost
   - **Date Lost**: When you last had the item
   - **Contact Preference**: How you want to be contacted

3. **Upload Photos** (Optional but recommended)
   - Clear photos help with identification
   - Multiple angles if possible
   - Close-ups of unique features

4. **Submit Report**
   - Review all information
   - Click "Submit Lost Item Report"
   - Receive confirmation

### Best Practices for Lost Items

- **Be specific**: More details increase chances of recovery
- **Act quickly**: Report as soon as possible
- **Check regularly**: Monitor for matching found items
- **Update status**: Mark as found when recovered

## Reporting Found Items

### Step-by-Step Process

1. **Navigate to Report Found Item**
   - Click "Report Found Item" from the main menu
   - Or use the quick action button

2. **Fill in Item Details**
   - **Item Name**: What you found
   - **Category**: Select category
   - **Description**: Detailed description
   - **Location Found**: Exact location
   - **Date Found**: When you found it
   - **Current Location**: Where item is stored

3. **Upload Photos** (Required)
   - Clear photos are essential
   - Multiple angles
   - Any identifying marks or features

4. **Submit Report**
   - Review information
   - Click "Submit Found Item Report"
   - Item goes to admin for approval

### Best Practices for Found Items

- **Secure the item**: Keep it safe
- **Report immediately**: Don't delay
- **Provide accurate location**: Help owner retrieve it
- **Respond to claims**: Check notifications regularly

## Searching for Items

### Basic Search

1. **Use the search bar** at the top of the page
2. **Enter keywords** related to your item
3. **View results** and click for details

### Advanced Filtering

Filter by:
- **Category**: Electronics, Documents, Personal Items, etc.
- **Date Range**: When item was lost/found
- **Location**: Specific buildings or areas
- **Organization**: Filter by organization
- **Status**: Active, Claimed, Returned

### Search Tips

- Use specific keywords
- Try different search terms
- Check multiple categories
- Expand date range if needed
- Contact admin if you think an item matches

## Managing Your Items

### View Your Items

1. Navigate to "My Items" in your profile
2. See all your reported items:
   - Lost items you've reported
   - Found items you've reported
   - Claims you've submitted

### Update Item Status

**For Lost Items:**
- Mark as "Found" when recovered
- Update description if needed
- Add additional photos
- Delete if no longer relevant

**For Found Items:**
- Update location if moved
- Add more details if needed
- Mark as "Claimed" when returned

### Item Lifecycle

```
Lost Item Flow:
Report → Active → [Matched] → Claimed → Returned → Closed

Found Item Flow:
Report → Pending Approval → Active → Claimed → Returned → Closed
```

## Claiming Items

### How to Submit a Claim

1. **Find the item** in search results
2. **Click "Claim This Item"**
3. **Provide verification details**:
   - Proof of ownership
   - Additional description
   - Serial numbers or unique identifiers
   - Purchase receipts (if available)
4. **Submit claim**
5. **Wait for admin review**

### Verification Process

Admins will:
- Review your claim details
- Compare with item description
- May request additional information
- Contact you for verification
- Approve or deny claim

### After Claim Approval

1. **Receive notification** of approval
2. **Coordinate pickup** with finder or admin
3. **Verify item** when collecting
4. **Confirm receipt** in the system
5. **Provide feedback** (optional)

## Privacy and Security

### What Information is Visible

**Public Information:**
- Item description (limited)
- Category and location
- Date lost/found
- Photos (may be blurred)

**Private Information:**
- Your full contact details
- Serial numbers
- Detailed descriptions
- Claim verification details

### Protecting Your Privacy

- Don't include sensitive information in public descriptions
- Use the messaging system for private details
- Report suspicious claims to admins
- Verify identity before meeting to exchange items

## Tips for Success

### For Lost Items

1. **Report immediately**: Don't wait
2. **Be detailed**: More information helps
3. **Check regularly**: New items are added daily
4. **Expand search**: Try different keywords
5. **Contact admin**: If you think you found a match

### For Found Items

2. **Report quickly**: Owner may be searching

### For Claims

1. **Provide proof**: Documentation helps
2. **Be specific**: Describe unique features
3. **Be patient**: Verification takes time

5. **Say thanks**: Appreciate the finder

## Troubleshooting

### Item Not Appearing

- Check if it's pending admin approval
- Verify all required fields were filled
- Contact admin if delayed

### Can't Find Your Item

- Try different search terms
- Expand date range
- Check multiple categories
- Contact admin for assistance

### Claim Not Processed

- Ensure all verification details provided
- Check for admin messages
- Allow time for manual review
- Contact admin if urgent

### Photo Upload Issues

- Check file size (max 5MB)
- Use supported formats (JPG, PNG)
- Ensure stable internet connection
- Try different browser if issues persist
