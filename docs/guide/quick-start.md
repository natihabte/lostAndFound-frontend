# Quick Start Guide

Get up and running with the Public Sector Lost & Found Management Platform in minutes.

## Overview

This quick start guide will walk you through:
1. Accessing the platform
2. Creating your first organization
3. Reporting a lost item
4. Reporting a found item
5. Processing a claim

## Step 1: Access the Platform

1. Open your browser and navigate to the platform URL
2. You'll see the landing page with language options
3. Select your preferred language (English, Amharic, Oromo, or Tigrinya)

## Step 2: User Registration

### For Regular Users

1. Click **"Sign Up"** on the homepage
2. Fill in your details:
   - Full Name
   - Email Address
   - Phone Number
   - Password
3. Click **"Create Account"**
4. Check your email for verification link
5. Click the verification link to activate your account

### For Organization Registration

1. Click **"Register Organization"** 
2. Fill in organization details:
   - Organization Name
   - Type (University, Government, Hospital, etc.)
   - Contact Information
   - Address
3. Submit for verification
4. Wait for admin approval

## Step 3: Report a Lost Item

1. **Login** to your account
2. Click **"Report Lost Item"**
3. Fill in the item details:
   - Item Name
   - Category (Electronics, Documents, Clothing, etc.)
   - Description
   - Location where lost
   - Date lost
   - Upload photo (optional)
4. Click **"Submit Report"**

Your lost item will be added to the database and searchable by others.

## Step 4: Report a Found Item

1. Click **"Report Found Item"**
2. Fill in the details:
   - Item Name
   - Category
   - Description
   - Location where found
   - Date found
   - Upload photo
3. Click **"Submit Report"**

The found item will be visible to users searching for lost items.

## Step 5: Search for Items

### Browse All Items
1. Click **"Browse Items"** on the homepage
2. Use filters to narrow down:
   - Category
   - Date range
   - Location
   - Organization

### Advanced Search
1. Use the search bar at the top
2. Enter keywords related to your item
3. Apply filters for better results

## Step 6: Submit a Claim

When you find an item that might be yours:

1. Click on the item card
2. Review the details carefully
3. Click **"Claim This Item"**
4. Provide verification details:
   - Proof of ownership
   - Additional description
   - Contact information
5. Submit the claim

## Step 7: Admin Functions

### Organization Admin Access

If you're an organization admin:

1. Navigate to `/admin/login`
2. Login with admin credentials
3. Access admin dashboard with:
   - User management
   - Item oversight
   - Organization settings
   - Reports and analytics

### Super Admin Access

Super admins can:
1. Manage all organizations
2. Approve organization registrations
3. System-wide configuration
4. Global reports

## Common Workflows

### Lost Item Recovery Process

1. **User reports lost item** → Item added to database
2. **Someone finds similar item** → Reports found item
3. **System suggests potential matches** → Notifications sent
4. **Owner submits claim** → Verification process begins
5. **Admin verifies claim** → Item marked as claimed
6. **Parties coordinate pickup** → Case closed

### Organization Onboarding

1. **Organization applies** → Registration form submitted
2. **Admin reviews application** → Verification process
3. **Organization approved** → Account activated
4. **Admin users created** → Access credentials provided
5. **System configured** → Branding and settings applied
6. **Training provided** → Users onboarded

## Troubleshooting

### Common Issues

**Can't login?**
- Check email verification
- Reset password if needed
- Contact admin for account issues

**Item not appearing?**
- Check if it's pending approval
- Verify all required fields were filled
- Contact admin if issues persist

**Claim not processed?**
- Ensure all verification details provided
- Check with organization admin
- Allow time for manual review

**Email notifications not working?**
- Check spam folder
- Verify email address in profile
- Contact admin to check email settings
