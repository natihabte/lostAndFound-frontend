# User Registration Guide

This guide covers the complete user registration and onboarding process for the Public Sector Lost & Found Management Platform.

## Registration Types

The platform supports two main types of registration:

### Individual User Registration
For people who want to report lost or found items within existing organizations.

### Organization Registration
For institutions that want to set up their own lost and found management system.

## Individual User Registration

### Step 1: Access Registration Form

1. Navigate to the platform homepage
2. Click **"Sign Up"** or **"Create Account"**
3. You'll be directed to the registration form

### Step 2: Fill Registration Form

**Required Information:**
- **Full Name**: Your complete legal name
- **Email Address**: Valid email for verification and notifications
- **Phone Number**: Contact number for claim coordination
- **Password**: Strong password (minimum 8 characters)
- **Confirm Password**: Re-enter password for verification

**Optional Information:**
- **Organization**: Select if you're affiliated with a registered organization
- **Department/Location**: Specify your department or building
- **Student/Employee ID**: If applicable to your organization

### Step 3: Email Verification

1. **Check your email** for a verification message
2. **Click the verification link** in the email
3. **Complete verification** by following the instructions
4. **Return to the platform** and log in

### Step 4: Profile Setup

After verification, complete your profile:

1. **Upload profile photo** (optional)
2. **Set notification preferences**
3. **Choose language preference**
4. **Configure privacy settings**

## Organization Registration

### Step 1: Access Organization Registration

1. Navigate to **"Register Organization"** on the homepage
2. Or visit `/organization-register` directly

### Step 2: Organization Information

**Basic Information:**
- **Organization Name**: Official name of your institution
- **Organization Type**: Select from dropdown (University, Government, Hospital, etc.)
- **Website**: Official website URL
- **Description**: Brief description of your organization

**Contact Information:**
- **Primary Contact Name**: Main administrator's name
- **Email Address**: Official contact email
- **Phone Number**: Main contact number
- **Address**: Complete physical address

### Step 3: Administrative Details

**Admin User Setup:**
- **Admin Name**: Primary administrator's full name
- **Admin Email**: Administrator's email (will become login)
- **Admin Phone**: Administrator's contact number
- **Admin Title**: Position/title within organization

**Organization Settings:**
- **Operating Hours**: When the lost & found service operates
- **Locations**: List of buildings/areas to be covered
- **Departments**: Relevant departments or divisions
- **Special Instructions**: Any specific procedures or requirements

### Step 4: Verification Documents

Upload required documents:
- **Official Registration**: Business license or registration certificate
- **Authorization Letter**: Letter authorizing the setup of lost & found service
- **Contact Verification**: Proof of contact information
- **Logo**: Organization logo for branding (optional)

### Step 5: Submission and Review

1. **Review all information** for accuracy
2. **Accept terms and conditions**
3. **Submit application**
4. **Wait for admin review** (typically 1-3 business days)

## Email Verification Process

### For Individual Users

1. **Registration Email**: Sent immediately after registration
2. **Verification Link**: Click to verify your email address
3. **Account Activation**: Account becomes active after verification
4. **Welcome Email**: Confirmation of successful registration

### For Organizations

1. **Application Received**: Confirmation that application was submitted
2. **Under Review**: Notification that review process has begun
3. **Additional Information**: Request for more details if needed
4. **Approval/Rejection**: Final decision on application
5. **Setup Instructions**: If approved, instructions for getting started

## Account Activation

### Individual Account Activation

After email verification:
1. **Login with credentials** you created during registration
2. **Complete profile setup** if not done during registration
3. **Explore the platform** and familiarize yourself with features
4. **Start using services** - report items, search, submit claims

### Organization Account Activation

After approval:
1. **Receive admin credentials** via secure email
2. **Access admin dashboard** using provided login URL
3. **Complete organization setup**:
   - Upload logo and branding
   - Configure locations and departments
   - Set up additional admin users
   - Configure notification settings
4. **Train staff** on platform usage
5. **Go live** and start accepting user registrations

## User Roles and Permissions

### Regular User Permissions
- Report lost items
- Report found items
- Search and browse items
- Submit claims
- Manage personal profile
- Receive notifications

### Organization Member Permissions
- All regular user permissions
- Access to organization-specific items
- Priority in search results within organization
- Organization-specific notifications

### Hall Admin Permissions
- All user permissions
- Approve found items in their location
- Assist with claim verification
- Access location-specific reports
- Manage local users

### Organization Admin Permissions
- All hall admin permissions
- Manage organization settings
- Create and manage hall admins
- Access organization-wide reports
- Configure organization branding

## Troubleshooting Registration

### Common Issues

**Email verification not received:**
- Check spam/junk folder
- Verify email address was entered correctly
- Request new verification email
- Contact support if issue persists

**Organization registration rejected:**
- Review rejection reasons provided
- Gather additional documentation
- Contact admin for clarification
- Resubmit with corrections

**Cannot access account after verification:**
- Verify password is correct
- Check if account requires admin approval
- Clear browser cache and cookies
- Try password reset if needed

**Profile information cannot be updated:**
- Check required field validation
- Verify file upload size limits
- Ensure proper image formats
- Contact admin for permission issues

## Best Practices

### For Individual Users

1. **Use institutional email** if affiliated with an organization
2. **Provide accurate information** for better matching and claims
3. **Keep profile updated** with current contact information
4. **Use strong passwords** and enable 2FA if available
5. **Verify email promptly** to avoid account delays

### For Organizations

1. **Prepare documentation** before starting registration
2. **Designate primary admin** who will manage the system
3. **Plan location structure** and admin hierarchy
4. **Review terms and policies** thoroughly
5. **Prepare staff training** for platform adoption

### Security Recommendations

1. **Strong passwords**: Use unique, complex passwords
2. **Regular updates**: Keep contact information current
3. **Privacy awareness**: Understand what information is shared
4. **Secure access**: Log out from shared computers
5. **Report issues**: Contact admin for suspicious activity

## Post-Registration Steps

### For New Users

1. **Explore the platform**: Familiarize yourself with features
2. **Set up notifications**: Configure preferences for updates
3. **Join organization**: If applicable, request to join your institution
4. **Start using**: Report items or search for lost belongings
5. **Provide feedback**: Help improve the platform with suggestions

### For New Organizations

1. **Complete setup**: Finish all configuration steps
2. **Train administrators**: Ensure admins understand their roles
3. **Create user guides**: Develop organization-specific instructions
4. **Promote platform**: Inform community about the new service
5. **Monitor usage**: Track adoption and address issues
